POST(URL, Array, function) → {Object}

POST request to the /users endpoint to create a new user. This request requires a request body with the fields: Username, Password, Email, Birthday. The fields are first validated against specified validators before the new user record is created. The new user in the MongoDB users collection hosted on Atlas will also be given a unique integer ID once it is created, giving JSON in this format: { ID: Integer, Username: String, Password: String, Email: String, Birthday: Date }
